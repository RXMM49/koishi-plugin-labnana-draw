# koishi-plugin-labnana-draw

一个为 Koishi 机器人设计的 Labnana AI 绘图插件，支持文生图和图生图（文带图）功能。

## 功能特性

- ✅ **文生图**：通过文字描述生成图片
- ✅ **图生图**：基于参考图片进行创作（文带图）
- ✅ **丰富的参数控制**：支持尺寸、步数、CFG、种子等参数调整
- ✅ **负面提示词**：排除不想要的内容
- ✅ **多模型支持**：可指定不同的AI模型
- ✅ **智能重试机制**：自动处理网络波动，最多重试3次
- ✅ **友好的错误提示**：清晰的错误信息和等待提示
- ✅ **性能监控**：显示生成耗时，便于优化

## 安装

```bash
npm install koishi-plugin-labnana-draw
```

或在 Koishi 控制台的市场中搜索并安装 `labnana-draw`。

## 配置

在 Koishi 控制台中配置以下参数：

| 参数 | 类型 | 必填 | 默认值 | 说明 |
|------|------|------|--------|------|
| apiKey | string | ✅ | - | Labnana API密钥 |
| baseUrl | string | ❌ | https://api.labnana.com | API基础URL |
| timeout | number | ❌ | 600000 | 请求超时时间（毫秒），默认10分钟 |
| proxyEnabled | boolean | ❌ | false | 是否启用代理 |
| proxyUrl | string | ❌ | - | 代理服务器地址（例如：http://127.0.0.1:1080） |

### 配置示例

```yaml
labnana-draw:
  apiKey: 'your_api_key_here'
  baseUrl: 'https://api.labnana.com'
  timeout: 600000
  proxyEnabled: false
  # proxyUrl: 'http://127.0.0.1:1080'  # 如果需要代理，取消注释并设置地址
```

## 使用方法

### 基本命令

#### 1. 文生图

```
.draw <提示词>
```

**示例：**
```
.draw 一只可爱的猫咪坐在窗台上
.draw 赛博朋克风格的城市夜景
```

#### 2. 图生图（文带图）

发送一张图片，然后使用：
```
.img2img <提示词>
```

**示例：**
```
[发送图片]
.img2img 变成动漫风格
```

#### 3. 检查API连接

```
.check
```

#### 4. 查看帮助

```
.help
```

### 高级选项

#### 文生图选项

| 选项 | 简写 | 说明 | 默认值 |
|------|------|------|--------|
| --negativePrompt | -n | 负面提示词 | - |
| --width | -w | 图片宽度 | 512 |
| --height | -h | 图片高度 | 512 |
| --steps | -s | 生成步数 | 20 |
| --cfgScale | -c | CFG比例 | 7 |
| --seed | - | 随机种子 | 随机 |
| --model | -m | 模型名称 | default |

**使用示例：**
```
.draw 美丽的风景画 -w 1024 -h 512 -n 模糊,低质量
.draw 角色立绘 -s 30 -c 10 --seed 12345
.draw 科幻场景 -m realistic -w 768 -h 768
```

#### 图生图选项

除了上述所有选项外，还支持：

| 选项 | 说明 | 默认值 |
|------|------|--------|
| --strength | 参考图强度（0-1） | 0.75 |

**使用示例：**
```
[发送图片]
.img2img 转换成水彩风格 --strength 0.5
[发送图片]
.img2img 增强细节 -s 40 --strength 0.8
```

### 实用技巧

1. **负面提示词**：使用 `-n` 排除不想要的元素
   ```
   .draw 公主 -n 丑陋,畸形,多余的手指
   ```

2. **固定种子**：使用 `--seed` 复现相同结果
   ```
   .draw 猫咪 --seed 42
   ```

3. **高质量输出**：增加步数和CFG值
   ```
   .draw 精美插画 -s 50 -c 12
   ```

4. **图生图强度调节**：
   - 强度接近 0：更忠实于原图
   - 强度接近 1：更自由发挥
   ```
   [发送图片]
   .img2img 轻微美化 --strength 0.3
   ```

## API说明

本插件调用 Labnana API 的两个主要端点：

- `/v1/images/generations` - 文生图
- `/v1/images/image-to-image` - 图生图

请确保您的 API 密钥具有相应的权限。

## 常见问题

### Q: 提示"API密钥无效"怎么办？
A: 请检查配置的 `apiKey` 是否正确，或使用 `.check` 命令测试连接。

### Q: 生成图片很慢？
A: AI绘图需要时间，通常需要10-60秒。插件已设置默认超时时间为10分钟，可以根据需要调整 `timeout` 配置。

### Q: 遇到网络错误怎么办？
A: 插件内置了智能重试机制，会自动重试最多3次（针对ECONNRESET、ETIMEDOUT等网络错误）。如果仍然失败：
- 检查网络连接是否正常
- 如果使用代理，确认代理配置正确
- 查看日志中的详细错误信息

### Q: 如何获得更好的生成效果？
A: 
- 使用详细、具体的提示词
- 适当调整步数（20-50之间）
- 使用负面提示词排除不良元素
- 尝试不同的种子值

### Q: 图生图时图片没反应？
A: 确保在发送图片后的同一条消息或紧接着发送 `.img2img` 命令。
